分析流程

0
用户输入
输入投资问题 + 选取博主参与者 + 配置讨论轮数
1
NewsAgent 市场资讯采集
读取本地资讯文件（output/news_input/），若无文件则由 LLM 自动生成市场资讯
输出：市场资讯全文
2
BloggerAgent 博主多轮讨论
每位博主基于市场资讯，围绕用户问题发表观点，进行多轮深度讨论
输出：各轮次讨论记录 + 共识总结
3
RiskAgent 风险评估
综合市场资讯和博主讨论结果，分析潜在风险因素，量化风险等级
输出：风险评估报告 + 风险等级（低/中/高）
4
DeciderAgent 综合决策
汇总资讯、讨论、风控三方输出，给出最终投资建议与决策依据
输出：投资决策报告（含建议、依据、风险提示）
LangGraph 状态机编排 AgentState 共享状态 SSE 实时流推送
